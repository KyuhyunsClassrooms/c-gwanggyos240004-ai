# 1. 프로그램 설계서 (5점)

## 1. 프로젝트 주제
학생 출석 관리 프로그램
## 2. 핵심 기능 목록
1.  학생 관리 – 등록, 조회, 수정, 삭제
2.  출석 관리 – 날짜별 출석 체크, 수정, 조회
3.  통계 기능 – 출석률 계산, 결석자 확인
4.  파일 저장/불러오기 – 데이터 유지
5. 메뉴 인터페이스 – 콘솔 메뉴로 기능 선택

## 3. 데이터 구조 설계
(프로그램에 필요한 핵심 변수, 배열을 설계하세요. 필요하면 구조체(struct)를 사용할 수도 있습니다.)

**방법 1: 배열 사용 (기본)**
```c
// 예시: 학생 5명의 이름과 성적 저장
char names[5][20];
int scores[5];
int count = 0;  // 현재 학생 수
```

**방법 2: 구조체 사용 (선택)**
```c
// 예시: 학생 정보를 구조체로 묶어서 관리
struct Student {
    char name[20];
    int kor;
    int eng;
    int math;
    float avg;
};
struct Student students[5];  // 학생 5명 저장
```
